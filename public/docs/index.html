Comfy — Proof Assistant for Formal Code Correctness

Comfy is a proof assistant designed for teaching formal code correctness to early computer science students. It lets you define inductive data types and recursive functions, then prove properties about them using calculation, structural induction, and proof by cases.

This documentation covers every part of the application: the setup screen, the declarations language, proof methods, and the rules you use to build calculation chains.
