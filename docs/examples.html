Worked Examples

These examples show complete proofs you can follow step by step in the Comfy UI. Each example shows the declarations, the goal, and every rule applied in the calculation chains.

Example 1: Pure Algebra

Prove x^2 + 2*x*y + y^2 = (x + y)^2 with no declarations.

Setup

  • Declarations: (empty)
  • Prove: theorem square_expand (x, y : Int) | x^2 + 2*x*y + y^2 = (x + y)^2

Proof

Method: calculation

  x^2 + 2*x*y + y^2
                              no forward steps needed

(x^2 + 2*x*y + y^2) =        backward: algebra recognizes this
  (x + y)^2

The built-in algebra decision procedure expands (x + y)^2 and confirms it equals x^2 + 2*x*y + y^2. One backward algebra step is all that's needed.

Example 2: List Length — 0 + len(xs) = len(xs)

A simple induction proof on lists.

Setup

Declarations:

type List
  | nil : List
  | cons : (Int, List) -> List

def len : (List) -> Int
  | len(nil) => 0
  | len(cons(a, L)) => 1 + len(L)

Prove:

theorem zero_plus_len (xs : List)
  | 0 + len(xs) = len(xs)

Proof

Method: induction on xs

Base Case: xs = nil

Goal: 0 + len(nil) = len(nil)

Method: calculation

  0 + len(nil)
= 0 + 0             defof len_1 => 0 + 0    — unfold len(nil) = 0
= 0                  = 0                     — algebra: 0 + 0 = 0

= 0                  undef len_1             — fold 0 back to len(nil)
  len(nil)

Forward: unfold len(nil) to 0, simplify 0 + 0. Backward: fold 0 back to len(nil). Chains meet at 0.

Inductive Case: xs = cons(a, L)

IH (named theorem): IH : 0 + len(L) = len(L)

Goal: 0 + len(cons(a, L)) = len(cons(a, L))

Method: calculation

  0 + len(cons(a, L))
= 0 + (1 + len(L))  defof len_2             — unfold len(cons(a, L)) = 1 + len(L)
= 1 + len(L)         apply IH                — apply IH: 0 + len(L) = len(L)

= 1 + len(L)         undef len_2             — fold back to len(cons(a, L))
  len(cons(a, L))

The key step is apply IH, which applies the induction hypothesis (0 + len(L) = len(L)) to replace 0 + len(L) with len(L). Note that the IH is a named theorem, not a numbered fact — you use apply IH rather than citing a fact number.
